```
Working Directory
| -- dir (ebm-sanskrit-word-segmentation)

```

To test with a particular feature vector use the tag of the feature while execution
* `python ebm_test_clique.py -t <tag>`

For example:
  * `python ebm_test_clique.py -t BM2`
